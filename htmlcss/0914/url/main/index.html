***경로 설정***

**절대 경로**

1. 최상위 디렉초리부터 순차적으로 경로 작성

2. 드라이브명(C:··· D:···)이나 프로토콜명(http://··· ftp://···)으로 시작

3. 기준 파일의 위치와 상관없이 대상 파일의 경로가 동일

4. 다른 사이트의 자원 사용 시

ex) <a href="http://www.naver.com">네이버</a> <img src="c:/omg/main.jpg" alt="">

네이버

**상대 경로**

* 1. 동일한 디텍토리에 있는 경우 *

- 파일명 확장자

ex) <a href="notice.html">공지사항</a> <img src="logo.png" alt="">

* 2. 하위 디텍토리에 있는 경우 *

- 폴더명/파일명,확장자

ex) <a href="html/notice.html">공지사항</a> <img src="images/logo.png" alt="">

* 3. 상위 디텍토리에 있는 경우 *

- /최상위폴더 : 최상위 디렉토리로

- ../파일명.확장자 : 한 수준만 위로

ex) <a href="../notice.html">공지사항</a> <img src="../../logo.png" alt="">

* 4. 복합 경로 *

- ../폴더명/파일명.확장자

ex) <a href="../html/notice.html">공지사항</a> <img src="../../images/icon/logo.png" alt="">
